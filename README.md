# Mantis
Workflow tool

Techs: 
- Based on Golang
- CLI with Cobra
- TUI with Gocui (awesome-gocui fork, check `https://pkg.go.dev/github.com/awesome-gocui/gocui` )
- Database with GORM + SQLite

Features:
- Terminal-based note taking and storing
- Cmds fuzzy search
